# PrivateLibraryList
蔵書管理リストアプリ

## 概要
個人の所有する書籍を管理するアプリ

## 主要な機能
書籍のタイトル、著者、出版社、購入日、読了日などを入力する

蔵書の一覧表示

タイトル、著者、未読・読了による検索

シリーズ作品をまとめて表示し、何巻まで購入したか（読み終えたか）を確認できる

## 制作方針
画面構成と機能の一部は、前回制作したTODO管理アプリを参考にする

## 制作上の課題
- 原作者・翻訳者など、著者に該当する人物が複数いる場合
- 巻数表記はないが複数の書籍が同一シリーズと見なされる場合
 
上記のようなケースに対応できるDBテーブルを設計すること
